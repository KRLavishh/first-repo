# Rock Paper Scissors

A best-of-ten built in both Java and Python. First to 6 wins takes the match.

## Features
- Best of ten match format
- Score tracking
- Personalized messages with player name
- Play again option

## How to Run

**Python:**
```bash
python3 rps.py
```


**Java:**
```bash
javac rps.java
java rps
```

- Python
